# 2D Top Down RPG
A tribute to my favorite childhood game, Dragon Warrior, to gain skills in c++.

## Prerequisites
- [SFML](https://www.sfml-dev.org/) 2.6.x
- [pkgconf](https://github.com/pkgconf/pkgconf)

### Install
```sh
brew install sfml@2 pkgconf
```

## Run
```sh
# run the game
make freshrun
```

## Controls
↑ ↓ ← → to move
